# What is Bing feeder?
The Bing feeder is a sprocket-driven feeder with closed-loop control. It was designed for the MicroSMT PNPv3,  it uses a simple high-level pulse trigger, so it can be directly controlled via OpenPnPâ€™s ReferenceAutoFeeder. Therefore, it is suitable for all types of PNP machines.


![image](https://github.com/microsmt/Bing-feeder/blob/main/picture30.jpg)


# Feature 1
Uses a photoelectric sensor to detect the sprocket position, achieving closed-loop control. The repeat position will never change.
![image](https://github.com/microsmt/Bing-feeder/blob/main/picture32.jpg)


# Feature 2
Through the elastic pad at the bottom, it can automatically adapt to tapes of different thicknesses and suppress tape vibration, preventing components from jumping out.
![image](https://github.com/microsmt/Bing-feeder/blob/main/picture33.jpg)

# Feature 3
A magnetic feeder mounting system makes feeder replacement quick and easy.
![image](https://github.com/microsmt/Bing-feeder/blob/main/picture34.jpg)
